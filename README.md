## 摘录笔记
- 村上春树《当我谈跑步时，我谈些什么》
```
1.和同什么人一起做什么事相比，我更喜欢一人默不作声地读书，或是全神贯注地听音乐。只需一个人做的事情，我可以想出许多许多来。
何解：慎独！孤独的妙处，在于直面最真实的自我。此处有真意，欲辨已忘言。
2.别人大概怎么都可以搪塞，自己的心灵却无法蒙混过关。违背了自己定下的原则，哪怕只有一次，以后就将违背更多的原则。
何解：自律！自发自觉的约束自己的言行，是知行合一的基本和最高境界。无论何时，何地。做到表里合一，言行合一。
俗世生活中，目之所及，小不忍失大者多矣！总有人无视红绿的约束，自由穿梭；总有人一言不合，暴力相向，凡此种种，不胜枚举。
千里之提，毁于蚁穴。有则改之无则加勉！
3.跑步成为我日常生活的一根支柱。只要跑步，我便感到快乐。在我诸多习惯里，跑步是最有益的一个。
何解：好习惯，多多益善。能觅得一例，持之以恒，足矣！
4.觉得“今天不想跑步”的时候，我经常问自己这样一个问题：你大体作为一个小说家在生活，可以在喜欢的时间一个人待在家里工作，既不需早起晚拥挤在满员电车里受罪，也不需出席无聊的会议。这不是很幸运的事儿么?与之相比，不就是在附近跑上一个小时么，有什么大不了的?
何解：自我暗示的借口，是自我驱动的力量源泉。借口总比理由多，学会设问，把借口变成正向的理由。
5.我超越了昨天的自己，哪怕只是那么一丁点，才更为重要。在长跑中，如果说有什么必须战胜的对手，那就是过去的自己。
何解：比较！最聪明的人，永远只和自己做比较。长跑中，需要战胜的自我。漫长的人生中，唯一需要战胜的也还是自我。
6.逼迫我在不喜欢的时间去做不喜欢的事情，我从小就无法忍受这一点。反之，倘若是我自己想做的事情，在自己想做的时间爱做多少就做多少，我会做得比别人更加卖力。
何解：兴趣！心之所向，甘之若饴。
7.当受到无缘无故的非难时，或是觉得能得到别人的接受却事与愿违时，我总是比平日跑得更远一些，让肉体更多地消耗一些，好重新认识自己是个能力有限的软弱人类。而且跑的距离长于平日，便是强化了自己的肉体，哪怕只是一点点。
何解：疗效！跑步是可以治愈坏情绪的。跑起来，排空大脑和身体的不开心，心灵会得到释放和自由。
8.忙就中断跑步的话，我一辈子都无法跑步。坚持跑步的理由不过一丝半点，中断跑步的理由却足够装满一辆大型载重卡车。
何解：借口！成功的人找方法，失败的人找借口。
9.发怒的话，就将那份怒气冲着自己发好了。感到懊恼的话，就用那份懊恼来磨炼自己好了。
何解：管理！情绪变化是最难捉摸的人体奇迹，情绪管理是最重要的自我修炼。
10.但凡值得一做的事情，自有值得去做甚至做过头的价值。
何解：真理！相信真理，许多疑惑不解，都是因为我们认知的局限。
11.人生基本是不公平的。这一点毋庸置疑。即便身处不公之地，我想还是可以追求某种公正。
何解：公平公正！真正的英雄，就是深刻认识到人生的无奈无常，还能够从容面对，不亢不卑走完一生的普通人。
12.我在自制的小巧玲珑的空白之中、在令人怀念的沉默之中，一味地跑个不休。这是相当快意的事情，哪还能管别人如何言说?
何解：自我！跑自己的路，哪管他人怎么说。
```
