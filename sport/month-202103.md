# 奔跑主题——奔跑需要理由吗？
```
记得有位酷爱跑步的日本作家，村上春树，在《当我谈起跑步，我在说些什么？》中说过：跑步不需要理由。想跑的时候，就迈开双腿。
如果做每一件事，都需要一个正当的理由驱动，或名正言顺的章程，那么这个世界就会缺失很多发明创造和意外惊喜的乐趣。
有一个悖论：人是很懒惰的动物。人性从根本上来说更容易受习惯的支配和影响。
习惯从哪里来？
这是一个如同先有鸡还是先有蛋一样玄妙的问题。
现代文明的缔造，离不开科技的发展，更离不开人类的创新发明。
我们创造了各种不同的国家制度，宗教，哲学思想，文学，法律法规；
各种不同的摩天大楼，丰富的产品：汽车，手机，笔记本，信息网络，高楼大厦等；
我们创造的法律法规等会来制约和规范我们的生活；
我们生产制造的各种产品也同样深刻影响我们的生活。
马克思说过：劳动力成为商品，是资本的本质，即人的异化。
是不是可以这样说：商品成为习惯，或者说商品消费成为惯性行为，是商品的本质，即物的异化。
商品包括各种各样，有形的无形的，生活消费用的，生产制造用的。
当代信息社会里，人人都离不开网络和手机。
走路看手机，容易受伤；开车看手机，可能会丢掉生命；
数字化时代，没有网络连接，没有网络共享，没有网络信息号，可能停电带来的危害更让人绝望。
扯得有点远，其实人类的很多行为，都会反过来影响人的成长和发展。
既然注定要收习惯的支配和影响，那就多培养一些好习惯吧。
跑步就是一个有益于身心健康的绝佳运动。
奔跑吧，从现在开始。
跑步，可以戒烟。
跑步，可以解酒。
跑步，可以...
一起发掘吧^_^
```
