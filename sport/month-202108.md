# 奔跑主题——跑量,配速,心率,步频,哪个更最重要？
```
跑步最关键的几个科学指标，一直没有标准的答案。
专家忙着著书立说，争论不休。
跑者们或各持己见，拥护所爱；或一头雾水，或偏信偏听。
我也简单谈一些自己身体力行的体会。
跑步，本质上是很个体，很独立，差异化很大的运动。
专业运动员，大众业余跑者，普通人（从不跑步）等，从科学的运动定义和区分，就有天壤之别。
普通人感觉到很吃力的速度和距离，对于大众业余跑者可能很轻松就能完成；
同样大众跑者竭尽全力的配速和距离，专业运动员也可以轻松写意的完成。
对于普通的大众业余爱好者，心率应该放在第一位。
尤其是年过三十，以减脂和健身为目的的跑者，更应该把心率控制在合理的范围内。
最有效的减脂心率，一般在个人最大心率的65~80%左右。
配速，步频，跑量，是相互关联，相互配合，相互促进的关系，不是矛盾对立的关系。
但是，这3个指标的提高，都有一个共同的前提条件，依赖于基础核心能力。
基础核心能力是配速，步频，跑量的坚实基础和强大后盾。
只有基础核心足够强大，才能有效的提高配速，步频，跑量。
什么是基础核心能力？
简单的说，基础核心能力就是跑步运动中需要的综合运动能力，包括：心肺相关呼吸能力，运动相关腰腹背肌肉能力，跑步相关腿部和臀部肌肉能力等。
基础核心能力可以通过跑步运动提高，更需要有针对性的无氧锻炼。
总之，强大的基础核心能力，是中长跑必备的能力。
当你遇到跑步瓶颈，速度快不起来，步频无法提高等，
做一些针对性的基础核心能力训练，会有奇效！
```
