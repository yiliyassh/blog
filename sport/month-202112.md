# 奔跑主题——奔跑成为一种信仰！
```
2021年底了，做一个年终总结。
全年跑量 2793 公里，时长 288.6 小时；
月均跑量 233 公里，时长 24 小时。
全年跑了13次半马，月均1次。
5公里最短用时 24分41秒，10公里最短用时 53分2秒，半马最短用时 1小时54分3秒。
对于一个跑步18个月的菜鸟小白，5公里从33分提高到了24分，10公里从1小时20分提高到53分，半马跑进2小时，也算是一点小小的安慰吧。
回顾1年半的跑步历程，不忘初心是关键。
有段时间，非常追求配速，一心只想着跑的越快越好，越快越能减脂，越快越能证明自己的强大。
1年的时间，配速从每公里7分提高到6分半，再提高到6分，再提高到5分半。
突然间，就遇到了瓶颈，无论如何努力，再也无法提高了。
跑步看似简单，一双鞋，一件短袖，一条短裤，迈开双腿就开跑了。
但是，想要跑的更远，更久，更健康，只有恒心和毅力，是远远不够的。
有目的的看了一些跑步相关的书籍和文章，也解答了心中的诸多疑惑：
配速和心率孰重孰轻？（心率更重要，不追求健康的人可忽略）
晨跑和夜跑，应该给如何选择？（我习惯晨跑）
空腹跑会伤害身体吗？（一家之言，我习惯空腹跑-10公里以内；半杯牛奶，吃个鸡蛋。）
......
基础核心力量也需要同步提高，否则无法支撑强度更大的跑量和更快的配速。
也知道了平衡匀速跑，才是中长跑的至简秘笈。
然后，调整了跑步的策略计划。
变更跑步的频率：每周至少休息一天，而不是每天都跑，从不缺席
跑步的类型：长久的持续一种匀速跑，不是最佳选择，需要多种类型混杂，如慢跑、匀速跑、燃脂跑、竞速跑、，LSD等。
习惯成自然，当跑步成为一种习惯，就没有什么可以阻挡你奔跑的步伐了。
习惯的力量，是肉体和精神的最完美结合。
当然，我们需要的是好习惯的正向循环。
当奔跑成为一种信仰，正向循环的好习惯，会给你远超预期的收益。
奔跑吧！
```
