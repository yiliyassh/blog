# 奔跑主题——一个月减5斤？
```
跑了一个月，体重下降了5斤多。
首先是很开心，其次是感觉很神奇，有点不可思议。
身体里的脂肪，真的是很奇妙的东西。
据说1公斤脂肪的面积，是1公斤肌肉的好几倍。
夏天快速减轻的体重里，大部分都是水分，小部分是脂肪，也有肌肉流失吧。
跑步神效初现，更增添了坚持的信心和动力。
加油，奔跑吧！
```
