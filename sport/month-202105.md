# 奔跑主题——奔跑会受伤吗？
```
跑步会不会受伤，这个问题我也曾经纠结了很久。
运动就会有风险。无论是跑步，还是走路。只是风险的大小和概率不同而已。
任何运动，过犹不及。
适量运动，科学运动，才能有效降低受伤的概率和风险。
首先要适量运动，以吃饭为例。
吃少了，会又还没饱的感觉，需要再吃一点；吃多了，会产生撑涨的感觉，需要停止进食。
运动也是一样，最直接的感觉永远是自己的身体。量少了可以加一点，量多了下次就减一点。
一定要相信我们最直接的感觉，我们的体感是最神奇最奇妙最值得信赖的朋友。
其次是科学运动。
如果说适量运动是感性体验和认知，那么科学运动则更强调方法和技巧的合理性和应用。
科学的运动理论，可以解释运动必要性和合理性，给出运动方法和运动建议等。
比如跑者最关心问题：
跑前要拉升吗？（必须）
跑量多少？（3km还是5km，或者更多？）
心率多少？（130，140,150？）
配速多少？（7分，6分，5分？）
步频多少？（160,170,180？）
跑后要拉升吗？（必须）
...
上述问题，其实没有标准答案。
每个人的体质都有差异，年龄和健康状况更是千差万别。
但有一些科学的定义，方法和公式可以参考：比如最大心率，最大摄氧量，慢跑，匀速跑，竞速跑，马拉松跑等。
我们需要通过不断的跑步实践，记录相关数据，定期分析总结（每日，周，月），找到一套最适合自己的跑步方案。
接下来的就是方案的执行，不停的奔跑，调整和改进方案，不断的优化迭代。
那么长期跑步而不受伤，是可以实现的。
```
